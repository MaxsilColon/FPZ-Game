# FPZ-Game

Un juego FPS de zombies desarrollado en Unity con sistema de compañero IA.

## Características

- ✅ Juego FPS completo con zombies
- ✅ Sistema de armas y munición
- ✅ Compañero IA que te sigue
- ✅ Movimiento fluido y controles intuitivos
- ✅ Efectos de sonido y música

## Controles

- **WASD** - Movimiento
- **Mouse** - Mirar alrededor
- **Click Izquierdo** - Disparar
- **Click Derecho** - Apuntar
- **R** - Recargar
- **Q** - Ataque cuerpo a cuerpo
- **Shift** - Correr
- **Espacio** - Saltar
- **C** - Agacharse

## Instalación

1. Clona este repositorio
2. Abre el proyecto en Unity 2022.3 o superior
3. Abre la escena principal en `Assets/Scenes/`
4. Presiona Play para jugar

## Herramientas de Desarrollo

El proyecto incluye herramientas de editor personalizadas:

- **Tools → FIX ALL PLAYER PROBLEMS** - Arregla problemas del jugador
- **Tools → CREATE PLAYER COMPANION** - Crea un compañero que te sigue
- **Tools → RESET EVERYTHING** - Resetea el juego al estado original

## Créditos

Desarrollado usando Unity Engine y assets de Easy FPS.
